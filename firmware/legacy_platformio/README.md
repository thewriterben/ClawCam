# Legacy PlatformIO Reference

This directory is reserved for intentional migration notes or extracted reference code from WildCAM_ESP32.

The active ClawCam firmware direction is ESP-IDF plus ESP-Claw-compatible components. PlatformIO/Arduino code may be used as a reference for camera, storage, power, and sensor behavior, but it should not become the long-term source of truth unless the project explicitly decides to support a legacy firmware line.
