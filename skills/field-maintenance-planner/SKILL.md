# Field Maintenance Planner

## Purpose

Turn ClawCam device health and detection context into a practical field-maintenance plan.

## Planning Inputs

| Input | Description |
|---|---|
| Device health | Battery, storage, last seen, errors, sensor status. |
| Deployment priority | Site importance, rare species activity, project deadlines. |
| Access constraints | Distance, route, weather, permission windows, and team availability. |

## Output Format

Produce a prioritized maintenance checklist with device IDs, locations or deployment labels, reason for visit, recommended supplies, and urgency.

## Safety Boundary

The planner may recommend actions, but it should not apply configuration changes, delete data, or trigger firmware updates without explicit approval.
